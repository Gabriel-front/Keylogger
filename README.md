## 📋 Sumário Executivo

Este relatório documenta a análise técnica de duas amostras de malware do tipo **keylogger**, identificadas durante atividade de *threat analysis*. Os artefatos foram submetidos à análise estática com o objetivo de mapear comportamentos maliciosos, vetores de exfiltração de dados e indicadores de comprometimento (IoCs).

> ⚠️ **Disclaimer:** Todo o conteúdo deste relatório é produzido para fins exclusivamente educacionais e de pesquisa em segurança ofensiva/defensiva. A execução destes artefatos em ambientes não autorizados é ilegal e antiética.
